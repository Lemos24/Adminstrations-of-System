# Теоретическое введение

**GRUB2 (GRand Unified Bootloader version 2)** — это основной загрузчик в большинстве современных дистрибутивов Linux. Он отвечает за загрузку операционной системы и предоставляет гибкие возможности настройки.

## Основные компоненты GRUB2:

- **/boot/grub2/grub.cfg** — основной конфигурационный файл (генерируется автоматически)
- **/etc/default/grub** — файл с основными настройками
- **/etc/grub.d/** — директория со скриптами для генерации grub.cfg
- **/boot/grub2/grubenv** — переменные окружения GRUB
- **/boot/grub2/device.map** — карта устройств

## Основные команды для работы с GRUB2:

- `grub2-mkconfig` — генерация конфигурационного файла
- `grub2-install` — установка GRUB2 в MBR или EFI
- `grub2-set-default` — установка ядра по умолчанию
- `grub2-reboot` — однократная загрузка указанного ядра
- `grub2-editenv` — редактирование переменных окружения
- `grub2-mkpasswd-pbkdf2` — создание пароля для GRUB

# Основные параметры /etc/default/grub

| Параметр | Описание | Пример |
|----------|----------|--------|
| `GRUB_TIMEOUT` | Время ожидания в секундах | `GRUB_TIMEOUT=5` |
| `GRUB_DEFAULT` | Пункт меню по умолчанию | `GRUB_DEFAULT=0` или `GRUB_DEFAULT=saved` |
| `GRUB_CMDLINE_LINUX` | Параметры ядра | `GRUB_CMDLINE_LINUX="quiet splash"` |
| `GRUB_GFXMODE` | Разрешение графики | `GRUB_GFXMODE=1024x768` |
| `GRUB_DISABLE_RECOVERY` | Отключение recovery режима | `GRUB_DISABLE_RECOVERY=true` |
| `GRUB_THEME` | Тема оформления | `GRUB_THEME=/boot/grub2/themes/system/theme.txt` |
