Теоретическая справка

Firewalld — динамический межсетевой экран

Основные компоненты: - Зоны (zones) — наборы правил для разных уровней доверия - Службы (services) — предопределённые правила - Богатые правила (rich rules) — сложные правила с условиями - Команда: firewall-cmd — основная для управления
