# Теоретическое введение

**RAID (Redundant Array of Independent Disks)** — технология виртуализации данных, которая объединяет несколько физических дисков в один логический для повышения производительности и/или отказоустойчивости.

## Основные уровни RAID:

- **RAID 0 (Striping)** — чередование данных для повышения производительности (без отказоустойчивости)
- **RAID 1 (Mirroring)** — зеркалирование для отказоустойчивости
- **RAID 5** — чередование с распределённой чётностью (минимум 3 диска)
- **RAID 6** — чередование с двойной чётностью (минимум 4 диска)
- **RAID 10** — комбинация зеркалирования и чередования

## Утилита mdadm:

`mdadm` — это инструмент для управления программными RAID-массивами в Linux. Она позволяет создавать, собирать, мониторить и восстанавливать RAID-массивы.

## Основные команды mdadm:

- `mdadm --create` — создание RAID-массива
- `mdadm --detail` / `mdadm -D` — детальная информация о массиве
- `mdadm --stop` — остановка массива
- `mdadm --assemble` — сборка существующего массива
- `mdadm --manage` — управление массивом (добавление/удаление дисков)
- `mdadm --monitor` — мониторинг массива
- `mdadm --examine` — просмотр суперблоков на дисках
- `mdadm --fail` — пометка диска как сбойного
- `mdadm --remove` — удаление диска из массива
- `mdadm --add` — добавление диска в массив

# Основные уровни RAID

| Уровень | Минимум дисков | Отказоустойчивость | Производительность |
|---------|----------------|---------------------|---------------------|
| RAID 0 | 2 | Нет | Высокая (чтение/запись) |
| RAID 1 | 2 | 1 диск | Высокая (чтение), средняя (запись) |
| RAID 5 | 3 | 1 диск | Высокая (чтение), средняя (запись) |
| RAID 6 | 4 | 2 диска | Высокая (чтение), низкая (запись) |
| RAID 10 | 4 | 1-2 диска | Очень высокая |

# Основные команды mdadm

| Команда | Назначение |
|---------|------------|
| `mdadm --create` | Создание RAID-массива |
| `mdadm --detail` | Детальная информация |
| `mdadm --manage --add` | Добавление диска |
| `mdadm --manage --fail` | Пометка сбойного диска |
| `mdadm --manage --remove` | Удаление диска |
| `mdadm --stop` | Остановка массива |
| `mdadm --assemble` | Сборка массива |
| `cat /proc/mdstat` | Просмотр состояния |
