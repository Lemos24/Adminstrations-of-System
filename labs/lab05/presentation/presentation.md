## Теоретическая справка

**Управление службами в Linux (systemd)**

Основные понятия:
- `systemctl` — основная команда для управления службами
- Юниты — единицы управления (сервисы, цели, сокеты)
- Зависимости — отношения между юнитами
- Цели (targets) — группы юнитов для определенных состояний системы
